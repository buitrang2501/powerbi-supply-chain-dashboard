# Refresh Instructions (Power BI)

1) Put the dataset file in:
   /data/Supply Chain & Sales Datasets.xlsx
2) Open the PBIX in /powerbi
3) Power Query:
   - Update the file path parameter (if used), or edit Source to point to /data
4) Refresh:
   - Refresh preview in Power Query
   - Close & Apply
   - Refresh visuals in report

Optional:
